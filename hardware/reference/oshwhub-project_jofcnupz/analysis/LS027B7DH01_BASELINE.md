# LS027B7DH01 L1 基线

## 已确认

- 型号：Sharp `LS027B7DH01`。
- 400 × 240、2.7 英寸反射式 Memory LCD。
- 模块：62.8 × 42.82 × 1.64 mm。
- 可视区：58.8 × 35.28 mm。
- 供电：VDD/VDDA 4.8–5.5 V，典型 5.0 V。
- 3.3 V ESP32 信号满足数据手册 VIH 最小 2.7 V 的名义条件，但仍需面包板和首板波形验证。
- 10-pin：1 SCLK、2 SI、3 SCS、4 EXTCOMIN、5 DISP、6 VDDA、7 VDD、8 EXTMODE、9 VSS、10 VSSA。
- EXTMODE 拉高到 VDD 时使用外部 EXTCOMIN；EXTCOMIN 为 1–20 Hz，并应低于画面刷新频率。
- VDD/VDDA 应同时上升或 VDD 先上升；下降时同时下降或 VDD 先下降。
- 数据手册外部电路图要求三颗电容并靠近显示连接器：DISP-VSS 0.1 µF、VDDA-VSS 0.1 µF 或更大、VDD-VSS 1 µF 或更大。

## 机械与连接器

- FPC 弯折区距玻璃边 0.8–6.0 mm。
- 最小内弯半径 R0.45 mm，禁止向偏光片侧反折，重复弯折不超过 3 次。
- 数据手册推荐顶接点/底接点 10-pin 0.5 mm 连接器，但不能仅凭间距选择；必须核对实物触点面和 Pin 1。

## 供应链状态

- 2026-07-22 查询时，DigiKey 将 `LS027B7DH01` 标为 Active，并显示 937 件现货、28 周标准交期。
- 同日 Mouser 页面将该型号标为 Obsolete。
- 这两个渠道状态冲突，说明生命周期必须标记为 `待验证`。用户已有实物，L1 原型不因此阻断；面向持续生产前必须确认制造商/授权渠道状态，并评估 Azumo/Sharp 后续兼容型号或锁定库存策略。

## L1 不得猜测的项目

- FPC 触点朝向、Pin 1 和自然弯折方向。
- 已购转接板是否真正 1:1 对应屏幕十根线。
- EXTMODE 最终固定方式与启动/关机期间状态。
- DISP 和 EXTCOMIN 的 GPIO 选择及异常恢复策略。
- IP5306 在显示静态低负载时是否会自动关断。
