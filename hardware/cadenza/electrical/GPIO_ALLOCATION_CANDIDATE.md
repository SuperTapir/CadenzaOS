# Cadenza GPIO 候选分配

状态：**L1/L2 联合规划候选，未冻结**。

| 功能 | 候选 GPIO | 依据/备注 |
|---|---:|---|
| Sharp SCLK | 48 | 复用参考板 TFT CLK；DevKit v1.0 板载 RGB LED 风险待验证 |
| Sharp SI | 12 | 复用参考板 TFT MOSI |
| Sharp SCS | 14 | 复用参考板 TFT CS |
| Sharp EXTCOMIN | 47 | 复用参考板 TFT DC |
| Sharp DISP | 39 | 复用原背光控制位置，移除 Q2/R13/R14 |
| B 键 | 5 | 复用旧按键输入候选 |
| Menu 键 | 8 | 复用旧按键输入候选 |
| 编码器 A 相 | 15 | 复用旧按键输入候选 |
| 编码器 B 相 | 16 | 复用旧按键输入候选 |
| 编码器按压 / A 键 | 17 | 复用旧按键输入候选 |
| 电源键 MCU 感知 | 18 | 运行时短按锁屏/解锁候选 |
| 硬件强制关机控制 | 6 | 候选；最终依电源电路验证 |

保留与限制：

- GPIO0/3/45/46 是启动配置脚，当前不用于新增人机输入或屏幕。
- GPIO19/20 保留给原生 USB D-/D+。
- N16R8 的 GPIO35/36/37 被模组内 Octal PSRAM 占用，不得分配。
- GPIO43/44 保留给参考板 UART0/CH340 下载调试路径。
- GPIO40/41/42 保留给 MAX98357 I2S 音频。
- GPIO1/2/21 暂保留参考麦克风接口；若 L1/L2 明确 DNP 麦克风，再决定是否释放。
- GPIO38 保留参考 RGB LED；是否保留灯本身可在 L3 决定。

这张表证明 GPIO 数量上有余量，不代表机械走线和电源键电路已经完成。最终冻结前必须同时复核屏幕面包板结果、编码器方向、USB、下载模式和电源键真机行为。
