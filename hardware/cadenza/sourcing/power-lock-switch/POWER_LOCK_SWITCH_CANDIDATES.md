# Cadenza 独立 Power / Lock 瞬时键采购候选（未冻结）

日期：2026-07-22  
范围：为 `mechanical/power-lock-candidates/` 的顶边 A、右侧边 B 两个位置寻找可由嘉立创 SMT 的常开瞬时键。本文不修改 KiCad、OpenSpec 或正式外壳，也不代表已选定最终型号。

## 结论

当前两个机械位置都是**从外壳边缘向内按**，因此直接实现应优先采用侧按（right-angle / side-push）开关：

1. 侧按可靠性主候选：G-Switch `GT-TC020A-H035-L1` / `C17179533`；
2. 侧按高库存低成本替代：SHOU HAN `TS24CA 250gf 013` / `C5373430`；
3. 顶按、Basic 采购主候选：XKB `TS-1187A-B-A-B` / `C318884`，仅适合改成正面/背面按压，或增加转向机构；
4. 顶按、高键轴机械替代：Kinghelm `KH-6X6X5H-STM` / `C2837531`，同样不能直接代替侧按。

如果维持现有顶边/右侧边按压方向，推荐先围绕 `C17179533` 做外壳手板，同时保留 `C5373430` 的第二 footprint 候选。`C318884` 虽然是 Basic 且库存显著更高，但方向不对，不能只因为便宜就硬套到当前壳体。

## 为什么 S1 是侧按首选

- 厂家图纸明确为 Horizontal Tact Switch，动作方向与顶边 A、右侧边 B 一致。
- 2025 年版精确型号图纸给出了本体、执行头和推荐 PCB land pattern。
- 1.8 N 比常见 1.0–1.3 N 小按键更不容易被握持误触，又没有 2.5 N 那么硬。
- 厂家图纸声明 100,000 次动作寿命；即便按 JLC 较保守的 50,000 次，也优于 S2 厂家给出的 20,000 次。

它仍不能冻结：`C17179533` 是 Extended、库存仅为几千到一万量级，而且厂家图纸与 JLC 寿命字段冲突。首板前应买样品，并在 BOM 冻结时重查库存和生命周期。

## 机械适配结论

### 顶边 A

- S1/S2 可把执行头直接朝向顶边，PCB footprint 靠近板边放置。
- S1 执行头中心仅约高出 PCB `0.95 mm`，现有圆筒导向和 Ø5.6 mm 按帽不能原样沿用；需要低位矩形推杆或带内侧凸台的 3D 打印键帽。
- S2 的执行头更宽、更硬，容易做防误触凹入键，但必须验证按帽侧向力不会撬动焊脚。

### 右侧边 B

- S1/S2 同样可以直接侧按，但必须重新检查与 EC12、右上/右下安装孔、后壳螺柱和握持区的组合间隙。
- 右侧边比顶边更容易在握持时受持续横向力，建议至少保留现有护栏高出按帽 `0.8 mm` 的设计思想。

### 顶按 T1/T2

- 只能直接实现垂直 PCB 的正面或背面按压。
- 若坚持边缘位置，需要 L 形转向推杆、小竖板或其它机构；这些都会增加卡滞、装配公差和零件数量，不属于当前低风险路线。
- 因此 T1/T2 是采购/版图备选，不是当前边缘方案的等价替换。

## Footprint 门禁

正式画封装前必须以本目录 PDF 的推荐焊盘为准，不能使用“4 脚轻触开关”泛用 footprint：

- S1 有两电气端子和两个机械定位孔/固定区域，推荐 land 宽度和孔位均为专用尺寸；
- S2 的焊盘、定位孔和外壳投影与 S1 不同，二者不是 drop-in 替代；
- T1 是四个焊脚、同侧两脚内部相连；它不是两组独立触点，不能拿来代替 Power/Lock 电路中理想的 DPST 隔离键；
- T2 虽也是四脚 SPST，但焊脚跨度与 T1 不同，同样不可共用 footprint。

如果希望一块 PCB 同时容纳 S1/S2，必须专门画组合候选封装并进行焊盘重叠、阻焊桥和贴装可制造性检查；在检查完成前不要假设可兼容。

## 电气边界

四个候选都是常开 SPST 瞬时键，额定值远高于 Power/Lock 感知电流；它们都**不能解决** IP5306 `KEY` 未公开低电平阈值、BAT54C 压降、USB 插入时关机语义、双击手电筒复用等问题。那些结论仍以 `electrical/power-lock/POWER_LOCK_DATASHEET_GATE.md` 的实测门禁为准。

## 冻结前验证清单

1. 各买至少 5 个 S1、S2 样品，确认实际包装、日期码和执行头方向。
2. 卡尺测本体、执行头宽高、未按下突出量及 PCB 表面到执行头中心高度。
3. 用 3D 打印顶边 A、右侧边 B 各一套低位推杆，验证 100 次按压不卡滞、不顶死开关。
4. 对每个位置做跌落/侧向挤压的低成本手板检查，焊脚不能承担按帽的持续侧向载荷。
5. 万用表确认未按开路、按下导通，并记录触点抖动供固件去抖设计使用。
6. 与 Power/Lock BAT54C 原型连测短按开机、短按锁屏、长按硬关机；未知电气行为继续标记待验证。
7. BOM 冻结当天重新核对 LCSC/JLCPCB 库存、Basic/Extended、PCBA 服务类型、MOQ 和是否需要夹具。
8. 只在样品、footprint、外壳手板全部通过后冻结型号和按键位置。
