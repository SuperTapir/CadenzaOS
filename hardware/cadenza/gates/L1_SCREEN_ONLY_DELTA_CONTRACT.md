# L1 Screen-only 最小修改合同

状态：**已冻结范围，电气与物理方向仍待验证。** 本合同只限定 L1 可以改什么；它不证明 FPC 方向正确、屏幕已经点亮或 PCB 已可生产。

基线为 `hardware/cadenza/working-base/`。新工作副本为 `hardware/cadenza/derived/l1-screen-only/`，建立时 39 个源文件哈希一致；其 PCB 与 `golden-import` PCB 逐字节一致。

## 1. L1 成功定义

L1 是可以提交少量 PCB 工程样机并 3D 打印装配外壳的最简 MVP。唯一功能变化是把原 ST7789 显示子系统替换为 Sharp LS027B7DH01，并调整屏幕安装结构。

L1 不包含按键重构、编码器、独立 Power/Lock、原生 USB、接口移位、麦克风改造或无关测试点。

## 2. 参考器件动作：5 删除 / 2 局部修改 / 70 保留

删除：

- `FPC1`：原 12-pin TFT FPC；
- `U6`：原 `3V3LCD` LDO；
- `Q2`、`R13`、`R14`：原背光驱动。

局部修改：

- `C20=10 µF, 25 V, X5R, C6119887`：0805 封装、料号和 GND 端保持不变；另一端从 `3V3LCD` 改为 `+5V`，并只允许移到新显示连接器旁作为 Sharp VDD 去耦；
- `C21=100 nF, 50 V, X7R, C5137467`：0805 封装、料号和 GND 端保持不变；另一端从 `3V3LCD` 改为 `+5V`，并只允许移到新显示连接器旁作为 Sharp VDDA 去耦。

两颗电容的额定电压有充足余量，容量按 Sharp 外部电路的标称值使用。机械 fit-check 证明新插座必须位于屏幕尾线中心，而不是旧 FPC1 的右侧位置，因此 C20/C21 可随显示接口移动，避免保留一条没有意义的长去耦回路。C20 缺少本地厂家 DC-bias 曲线，但 10 µF 标称值相对 VDD 的 1 µF 要求余量很大；C21 的 100 nF 是标称值，不能被表述为容差后仍保证不少于 100 nF。

其余 70 个参考器件全部保留。它们的 designator、值、料号、封装、坐标、旋转、板面和焊盘网络必须保持不变；KiCad 序列化可能改变整文件字节，因此门禁比较规范化对象属性，不要求整个文件字节相同。

## 3. 新增器件：仅 2 个

- `J_DISP1`：Hirose `FH34SRJ-10S-0.5SH(50)` / `C324723` 双接触连接器及已复核 footprint；PCB 旋转和 Pin 1 最终实物方向仍须通过 1:1 门禁；
- `C_DISP1=100 nF`：DISP 到 GND，靠近 J_DISP1 pin 5。

不增加显示测试点、0 Ω 配置电阻或焊桥。测量优先使用连接器焊盘和现有可达节点。

## 4. Sharp 唯一电气引脚合同

| Sharp pin | 信号 | L1 网络 |
|---:|---|---|
| 1 | SCLK | GPIO39 |
| 2 | SI | GPIO48 |
| 3 | SCS | GPIO47 |
| 4 | EXTCOMIN | GPIO14 |
| 5 | DISP | GPIO12，并经 C_DISP1 100 nF 到 GND |
| 6 | VDDA | +5V，邻近 C21 100 nF |
| 7 | VDD | +5V，邻近 C20 10 µF |
| 8 | EXTMODE | +5V |
| 9 | VSS | GND |
| 10 | VSSA | GND |

## 5. 参考网络动作：7 删除 / 8 修改 / 87 保留

删除 7 条旧显示网络：`3V3LCD`（其 C20/C21 端点并入既有 `+5V`）、`GPIO3`、`Net-(Q2-B)`、`Net-(Q2-C)` 以及 FPC1 pad 6/13/14 的三个单端网络。U4 pin 15（GPIO3）在 L1 变为未使用，不重新分配。

只允许修改 8 条共享/复用网络：

- `GPIO39` → SCLK；
- `GPIO48` → SI；
- `GPIO47` → SCS；
- `GPIO14` → EXTCOMIN；
- `GPIO12` → DISP；
- `+5V`：只增加 Sharp pin 6/7/8、C20/C21 电源端和必要局部支路；
- `GND`：只删除旧显示端点并增加 Sharp pin 9/10、C_DISP1 和显示局部接地；
- `3V3M`：只删除 U6 pin 3 端点。

其余 87 条参考网络的规范化端点集合保持不变。特别是 `/VOUT`、GPIO18、GPIO6、U1.KEY、GPIO19/20 和所有 USB 网络不改。

## 6. PCB 几何边界

- 板框、四个安装孔以及 70 个非显示器件全部锁定。
- C20/C21 的器件身份、料号和封装锁定，位置只允许移入 J_DISP1 邻近去耦簇。
- 五条复用 GPIO 仍使用原有五个引脚，但按 PCB 可布线顺序重新分配 Sharp 信号；这只是固件 pin map 变化，不增加跳线或新 GPIO。
- USB1、U3、R1/R2 等 USB 器件坐标和网络端点冻结；若 J_DISP1 下方需要避让，只允许重做与其相交的 D+/D−/CC1 局部铜，并须通过差分连续性和 DRC 门禁。
- `+5V` 与 GND 是共享网络，不允许整体重布。只增加显示区局部支路；GND zone 边界保持不变，重新填充后须比较非显示区域。
- 旧 `3V3LCD`、背光和 GPIO3 的专用铜可删除。

## 7. 明确冻结的子系统

- USB/下载：USB1、CH340C U3、Q1、D+/D−、DTR/RTS、UART0 和 USB-C 坐标；
- 电源：IP5306、TLV62568、SW1、U1.KEY、`/VOUT` 与其全部功率铜；
- 输入：KEY1–KEY6、SW2、BOOT1、RST1；
- 音频：MAX98357A U5、扬声器接口及 GPIO40/41/42；
- 存储：CARD1 及 GPIO9/10/11/13；
- 其他：ESP32-S3 模组、麦克风、RGB LED、板框和安装孔。

## 8. 仍待验证，未因此扩大 L1

- 最终板载 10-pin FPC 连接器料号、上/下接点和 3D 模型；
- 面包板点屏、EXTCOMIN、供电时序和实测电流；
- 屏幕 FPC 的真实折弯路径、锁扣空间和外壳闭合；
- 完成后的 ERC/DRC、原理图↔PCB、Gerber/BOM/CPL、嘉立创预览和首板实测。
