# Cadenza D1 Rev A 硬件、外壳与装配 Draft v50

> 状态：制造候选，不是无条件下单许可。本文是 Rev A 当前权威入口；其他文档与本文冲突时，以本文和 `kicad/production-v50-release-20260721/` 中的工件为准。

## 1. 最终愿景

Cadenza D1 是以听、记、浏览和轻量交互为核心的口袋设备，而不是复刻掌机。它吸收 Playdate/T-Embed 已证明舒适的横向握持逻辑，但采用自己的输入语言：左侧 `Menu` 与 `B`，中央 2.7 英寸半反透 Memory LCD，右侧为同心交互件，外环旋转、中心 `A` 按压。设备还应具备持久化存储、灵敏麦克风、可用的扬声器、Wi-Fi、Bluetooth、USB-C、锂电池供电和实体总电源开关。

| 项目 | Playdate 的启发 | Cadenza D1 Rev A |
|---|---|---|
| 显示 | 高可读性单色反射屏 | 400 x 240、2.7 英寸 Sharp Memory LCD，优先户外和常亮可读性 |
| 输入 | 方向键、A/B、侧面曲柄 | Menu、B、同心滚轮+A；不做方向键 |
| 无线 | Wi-Fi、Bluetooth | ESP32-S3 模组原生 Wi-Fi/BLE |
| 音频 | 单声道扬声器、麦克风 | 数字 MEMS 麦克风、独立功放与更大声腔 |
| 工业设计 | 方正、薄、强识别度 | 横向圆角主体加背部多角度棱面，不复制 Playdate 外观 |

## 2. 已冻结的 PCB 基线

- PCB：116.0 x 64.0 mm，厚 1.6 mm，4 层。
- 叠层：`F.Cu / GND.g1 / POWER.g2 / B.Cu`。
- 板标：`CADENZA D1 REV A`、`HW V50 2026-07`。
- 最小线宽 0.15 mm；最小实测间距 0.1516 mm；最小钻孔 0.20 mm。
- 主控：ESP32-S3-WROOM-1-N16R8，提供 Wi-Fi/BLE；板上包含 USB-C、锂电池充电/电源路径、数字麦克风、I2S 功放、microSD、LCD、按键和旋转编码器。
- KiCad DRC：0 violation、0 unconnected、0 footprint error。
- KiCad ERC：0 violation。
- PCB 比原理图多 8 个有意的机械对象：`LCD1`、`H1-H4`、`FID1-FID3`，不是漏画的电气元件。

背部棱面应形成至少两个稳定桌面姿态，推荐约 12-15 度阅读角和 28-35 度录音/桌钟角。棱面不得侵入电池膨胀余量、天线禁布区、扬声器背腔和连接器插拔空间。

## 4. 组件与外壳安装关系

| 区域 | PCB/组件要求 | 外壳配合 |
|---|---|---|
| LCD | 正面中央，FPC 按实物自然弯折进入连接器 | 屏窗只压玻璃非显示区；禁止压 FPC；预留 0.3-0.5 mm XY 余量 |
| Menu/B | 正面左侧独立按键 | 独立键帽、防串键导向柱，键程和触发力由样机确定 |
| 滚轮+A | 正面右侧同心结构 | 外环驱动增量编码器，中心独立按压 A；止口承担径向载荷，焊点不承力 |
| 麦克风 | 左上，远离扬声器和开关电感 | 声孔同轴、短密封声道，用泡棉/硅胶圈隔绝壳内漏声 |
| 扬声器 | 服从 PCB 实际位置 | 独立密闭后腔、前向孔阵列、泡棉圈密封，不能裸贴在壳上 |
| USB-C | 底边或侧边 | 插头全插入无干涉；壳体承担插拔力 |
| 总电源 | 侧边 | 滑帽行程匹配并标记 ON/OFF，避免口袋误拨 |
| microSD | 侧边可维护位置 | 留退卡行程和手指槽，不拆壳即可换卡 |
| 电池 | PCB 背面大平面区 | 软包电池用拉带和低应力胶固定；四周至少 0.5 mm，厚度方向留膨胀余量 |
| 天线 | ESP32 模组天线端靠壳边 | 天线上下和前方无铜、无电池、无扬声器磁体、无金属喷涂 |

## 5. 屏幕冻结规则

用户已有的 2.7 英寸 400 x 240 Sharp Memory LCD 是唯一不可随意替换的外购件。照片上的 FPC 标识可用于追溯，但不能单凭照片冻结脚序。量产前必须由准确料号/卖家引脚表或同批屏实测确认：

- FPC 触点数、间距、接触面方向、插入方向和加强板厚度。
- `VDD`、`VDDA`、`GND`、`SCLK`、`SI`、`SCS`、`EXTCOMIN/DISP` 等真实脚序。
- 连接器上下接触类型和锁扣方向。
- 屏幕外形、玻璃厚度、有效显示区和 FPC 出口尺寸。

在这些项目确认前，`LCD FPC 连接器封装冻结` 与 `前壳屏窗冻结` 都是阻塞项。不要给裸 FPC 夹杜邦线或在未知脚序下上电。

## 6. 音频要求

麦克风灵敏度不能只靠换型号解决。数字 MEMS 麦克风声孔不得被焊膏/胶遮挡；外壳声道应短直且气密；位置远离扬声器、功放、电感和高速时钟；固件提供可调增益与 AGC/降噪基线。

扬声器低频能力受振膜面积、背腔和密封共同影响：采用与功放匹配的 8 ohm 薄型扬声器；背腔与主机内部隔离；泡棉圈封闭泄漏缝；出音孔总面积建议不低于振膜有效面积约 20%。定壳前应实测 3 个扬声器候选和 2-3 个声腔深度，不能把标称功率当作音量保证。

## 7. BOM 与装配策略

`kicad/production-v50-release-20260721/assembly/cadenza-d1-v50-jlc-core-bom.csv` 与 `cadenza-d1-v50-jlc-core-cpl.csv` 已严格配对，均覆盖 60 个核心贴片位号。常用 0603 电阻/电容已冻结为高库存精确 LCSC 料号。`R4` 由 1.18 kohm 调整为 1.2 kohm，设定值变化约 1.7%，仍需在电池测试中复核阈值。

8 个机械/接口位号已冻结在 `cadenza-d1-v50-hand-frozen.csv`：`J4/J5 = C160352`、`J6 = C160405`、`SW1 = C221660`、`SW2-SW5 = C269272`。这些仍作为首板手焊/实物试装件，不计入 60 位号核心 SMT 上传。

手工装配电气器件已经全部冻结：`FB1/FB2` 使用 Murata `BLM18KG601SN1D`（LCSC `C85833`）；`L1/L2` 使用 Murata `DFE322520FD-1R0M=P2`（LCSC `C439775`）；`R73` 使用 TT Electronics `LRMAP2512-R01FT4`（LCSC `C5116354`）。其中 L1 从 1.2 uH 收敛为 1 uH，属于 TPS63070 数据手册明确支持的电感值；同一颗 1 uH 电感也满足 TPS61023。完整冻结清单见 `assembly/cadenza-d1-v50-hand-frozen.csv`，`hand-tbd.csv` 现为空表。首板仍建议先手焊这些机械/高电流器件并做样件装配确认，不应在确认前直接转为批量 SMT。

- 优先选嘉立创/LCSC 有库存且数据手册、封装、高度可核验的器件。
- 嘉立创没有合适件时允许淘宝或其他分销商采购并手焊，但须先买样件量尺寸、核脚位和耐流。
- 功率电感、磁珠和精密电阻不能仅按尺寸替代；须核对饱和电流、温升电流、DCR、阻抗曲线和容差。
- 两面都有器件；完整代贴需双面/标准 SMT。经济型单面贴装仅适合“核心顶层代贴 + 其余手焊”。

## 8. 嘉立创制造与导入

PCB 下单上传 `kicad/production-v50-release-20260721/cadenza-d1-v50-gerbers.zip`；若流程未自动识别钻孔，再补 `cadenza-d1-v50-drill.zip`。首板建议 4 层、1.6 mm、FR-4、外层 1 oz、ENIG。报价页须再次确认 0.20 mm 成品孔、0.15 mm 线宽/间距和板边连接器工艺能力。

嘉立创 EDA Pro 导入步骤：

1. 选择 `导入 -> KiCad`。
2. 上传 `kicad/production-v50-release-20260721/cadenza-d1-v50-kicad-import.zip`。
3. 选择“导入文档并提取库”，保留源文件样式；自动关联的封装和 3D 模型不能直接视为已验收。
4. 新建独立项目 `cadenza-d1-v50`，不要覆盖 `v38` 导入记录。

导入后逐项核验：板框 116.0 x 64.0 mm；Top/GND/POWER/Bottom 四层；90 个 PCB 对象、4 个安装孔、3 个基准点和 LCD 包络；USB-C、microSD、LCD FPC、开关和滚轮方向；GND 覆铜、过孔、天线 keepout、丝印和阻焊开窗；最后重新运行嘉立创 DRC。任何自动器件标准化/封装替换都要人工确认。

当前在线验收状态：导入包已经生成；此前 `v38` 已进入嘉立创 EDA Pro。2026-07-21 自动化复核时编辑器 WebGL 画布持续空白，因此 `v50 在线导入视觉验收` 仍待完成，不能标记为已通过。

## 9. 外壳 CAD 基线

参数化草模为 `mechanical/cadenza-d1-enclosure.scad`。参考外壳包络 122 x 74 mm；PCB 居中，参考原点 `[3.0, 5.0]`。外壳拆分为前壳、后壳、滚轮外环、A 中心键、Menu/B 键帽、开关滑帽和可选扬声器压框。

`SW1`、`J1`、`J4/J5`、`U2`、`U5` 已按冻结的 LCSC 料号固化为 5 个本地 STEP，并通过 `${KIPRJMOD}` 相对路径覆盖 6 个关键位号。KiCad 已重新导出约 10 MB 的装配 STEP，双面渲染确认 USB-C、侧滑开关、JST 和 QFN 均位于正确板面且未见镜像或明显偏位。真实装配 STEP 与 19 mm 外壳相交时只发现 J4/J5 顶插 JST 压入后盖；将后壳深度改为 14.5 mm、总厚度改为 21.5 mm 后，器件碰撞归零，只剩安装柱与 PCB 支撑面的零厚度接触。模型证据不能替代实物尺寸：仍须检查 FPC 锁扣、USB 插头、microSD 退卡、滚轮间隙以及电池/扬声器包络。

首轮先 FDM 打印尺寸验证壳，再做树脂/尼龙手感壳。尺寸壳必须验证四孔装入、屏幕无受压、插拔件可操作、按键不串键、滚轮不卡、电池不受压、扬声器密封和两个桌面角度稳定。
