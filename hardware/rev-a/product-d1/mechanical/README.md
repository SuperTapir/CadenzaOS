# Cadenza Rev A D1 外壳与装配 Draft

状态：机械接口草案，不能直接下单。单位均为毫米。

本目录把 PCB、屏幕、右侧滚轮子板、声学件与外壳放进同一个坐标契约。目标不是先画一个“像产品”的壳，而是让最终 STEP/STL、PCB 和实物装配使用同一组基准，避免在嘉立创导入后靠肉眼挪位置。

## 1. 产品构型

- 名义外形：`122 x 74 x 21.5`，横向掌机比例，四角切面而非纯圆角。真实 PCB STEP 碰撞检查证明 19 mm 版本会压住 J4/J5 顶插 JST；21.5 mm 版本在后盖内表面保留约 1.2 mm 净空。
- 正面中央：夏普 `LS027B7DH01`，面板 `62.8 x 42.82 x 1.64`，有效区 `58.8 x 35.28`。
- 正面左侧：独立 `Menu`、`B`；下方为独立密封扬声器出音区；左下小孔为麦克风。
- 正面右侧：直径不超过 `28` 的滚轮，按下滚轮就是 `A`，不再额外放一个 A 键。
- 右侧滚轮由 `34 x 38 x 1.6` 六线子板承载；前壳承受按压载荷，四颗 M2 螺丝只负责定位和防转。
- 边缘：底部 USB-C、右侧 microSD、左侧保持型电源开关。
- 背壳：可拆卸，外部棱面/支脚候选提供约 `18°` 和 `35°` 两种桌面姿态。

## 3. 参数化 CAD

[`cadenza-d1-enclosure.scad`](./cadenza-d1-enclosure.scad) 的顶部参数是机械冻结入口。`part` 可设为：

- `front`：前壳、屏幕口、Menu/B、滚轮、麦克风和扬声器孔。
- `rear`：背壳、主板柱、USB/SD/电源开口及桌面切面候选。
- `knob`：EC12 的 6 mm D 轴滚轮帽。
- `assembly`：装配视图，并显示 PCB、屏幕和电池碰撞包络。
- `exploded`：爆炸视图。

OpenSCAD 可分别导出 STL；给 CNC/注塑厂前应在 FreeCAD 或其他实体 CAD 中转 STEP，并重新检查实体、壁厚和公差。首轮 FDM 建议 `0.2 mm` 层高、4 层壁、PETG，按 `0.25 mm` 单边装配间隙起步。

## 4. 安装顺序

1. 将屏幕从前壳背面放入四点柔性支撑，不允许螺丝或硬柱压玻璃；四周用 `0.3-0.5` 泡棉限制窜动。
2. FPC 保持自然弯曲并进入主板 J2，不折死角，不让背壳或电池压住排线。
3. 将 EC12 穿过前壳止口并锁紧轴套，再安装右侧子板；插入六线防呆线束。
4. Menu/B 使用独立键帽和低矮轻触开关，键帽周边保留 `0.25-0.35` 单边间隙。
5. 麦克风底部声孔对准独立硅胶声道；硅胶同时贴合 PCB 和前壳，隔绝机内及扬声器前腔。
6. 扬声器用泡棉圈密封到前壳格栅，背面安装独立可替换声腔，不与麦克风共腔。
7. 安装主 PCB、USB/SD/电源开关，再接扬声器和电池；电池用可移除胶条固定，不用永久灌封。
8. 合背壳前检查天线净空：外壳坐标
   `X=94.75..119.05, Y=51.75..69.25` 前后不得有电池、扬声器磁钢、
   螺丝嵌件或导电涂层；塑料棱面可以穿过，但不得镀金属。

## 5. 仍需实物冻结的输入

下列任何一项未完成时，外壳只能标记为 Draft：

- 屏幕 FPC 接点面、pin 1 和 J2 实际插入方向；用实物和 1:1 图确认，不能仅依赖网图。
- 电池长宽厚、保护板凸起、导线出口、JST 极性和允许充电电流。
- 扬声器盆架直径、厚度、磁钢、焊片和振膜最大行程；同步选择 `1.5/2.0/3.0 cm3` 声腔。
- Menu/B 开关总高、键帽行程和手感。
- USB-C、microSD、保持型电源开关焊接后相对板边的真实位置。
- 完整样机重心；据此把背面两组切面修正到稳定、不遮挡扬声器且不压按键的桌面角度。

## 6. 机械发布门

1. 以卡尺测量所有样品，并把结果回填到 SCAD 顶部参数。
2. 打印 1:1 前壳薄片，验证屏幕、滚轮、Menu/B、接口开口和螺丝孔。
3. 打印完整空壳，做 FPC 弯折、电池和线束装配演练。
4. 装入工作 PCB，验证滚轮按压载荷不传给焊点，USB/SD 可插拔，麦克风与扬声器不漏气串腔。
5. 通过跌落、桌面两角稳定性、热、音量、啸叫、Wi-Fi RSSI 和拆装寿命测试后，才导出带版本号的生产 STEP/STL。

## 7. 数字碰撞证据

- `production-v50-release-20260721/mechanical/cadenza-d1-v50-pcb-aligned.stl` 是由 KiCad 装配 STEP 按板框基准转换到外壳坐标的真实器件网格。
- `cadenza-d1-collision-review.scad` 分别计算该网格与前壳、后壳的交集。
- `check-collision-v50.sh` 要求前壳交集为空；后壳只允许四个安装柱在 `Z=10.2` 支撑面产生零厚度共面接触。
- 该检查关闭数字模型碰撞门槛，但电池、扬声器、键帽、FPC 和线束仍必须以实物测量冻结。
