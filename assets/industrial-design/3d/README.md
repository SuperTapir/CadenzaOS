# Cadenza 3D 概念模型 V1

这是用于讨论外观比例、握持和器件包络的参数化 OpenSCAD 概念模型，不是可直接开模、CNC 或装配的生产 CAD。

## 当前包络

- 主机本体概念尺寸：`120 × 50 × 16 mm`
- 一体式实心后壳最大凸出：约 `10 mm`
- 中央最大总厚度：约 `26 mm`；四周边缘仍保持圆润纤薄
- 固定屏幕玻璃：`62.8 × 42.82 × 1.64 mm`
- 屏幕可视区：`58.8 × 35.28 mm`
- 旋压 `A` 控制器面径：约 `20.5 mm`
- 屏幕玻璃包络隐藏在前壳下；外露黑色窗口约 `60.0 × 36.5 mm`，可视区外只留约 `0.6 mm` 细黑收口
- 正面以屏幕中心线为整机中轴，屏幕几何居中；左右控制区宽度完全相等
- `B` 面径约 `10.5 mm`；`Menu` 为约 `8.5 × 3.4 mm` 的低矮胶囊键；旋压 `A` 约 `20.5 mm`

模型包含屏幕、`B`、`Menu`、旋压 `A`、顶部金属 `Power/Lock`、向机顶中央收近的 USB-C 充电口、机顶扬声器孔、屏幕凹槽上侧内壁中央的隐藏式麦克风微孔、机底右下角 MicroSD 和干净的可维护后盖。USB-C 与 MicroSD 分属上、下两边，避免同时插拔干涉。屏幕玻璃的安装黑边由珍珠银白前壳覆盖，屏幕表面比前壳低约 `0.5–0.7 mm`，正面只显示窄黑色窗口收口。`Menu` 键不再使用文字，改为键帽上的三点 `•••` 标识。当前不包含扩展坞、背板导轨或外接模块。

背面采用 Sony / 日系仪器方向：大面积安静表面中嵌入一块居中的长圆角电池与维修盖，只保留窄分件缝、四条开启防滑纹和一个低调紧固点。该结构借鉴 SONIC SticKURU Timer 背面“功能细节集中、其余表面克制”的秩序，不复制其具体电池仓几何。

背面采用一体式实心楔形后壳：从主机后壳的宽大根部连续收窄到外侧平台，约凸出 `10 mm`。侧面轮廓内全部为实体，没有空隙、背夹、支撑杆、翻板、转轴或活动支架。斜线是实心后壳的外轮廓，不是独立零件。外表面保留居中长圆角维修盖板和极少服务细节。它通过宽而顺滑的过渡与主体融合，避免成为额外挂载的“书包”。各摆放姿态的重心、摩擦力和按键操作稳定性必须由第三方用真实重量样机验证。

维修盖中央使用浅浮雕的花体 `Cadenza` 字样，同材质、同颜色，主要依靠侧光阴影显现，不做高对比印刷。

机底不使用独立深色橡胶条，改为两条与外壳一体成型、同色且仅轻微突出的纵向小棱。小棱沿机身前后厚度方向延伸，不沿机身左右长边横置；其轮廓必须完整收在机底银白侧壁内，不跨越与楔形后壳的交界线。实际防滑性必须结合表面纹理和样机摩擦测试确认。

机顶长圆扬声器开孔的内侧覆盖一整片深灰色细密声学防尘网。麦克风不再使用正面可见黑色圆孔，而是在屏幕凹槽上侧内壁中央设置约 `0.8 mm` 的侧向隐藏微孔，正视时由前壳保护唇遮挡。该微孔只定义外观拾音口；实际麦克风单元可通过密封声道横向引开，并与扬声器独立音腔和壳体振动软隔离。第三方必须根据扬声器频响、声压和防尘要求选择网材、目数、声阻与固定工艺，不能使用会明显衰减音量或高频的厚滤网。

当前 CMF 概念以 `PANTONE Cool Gray 2 C` 作为正面塑料按钮浅灰参考，以 `PANTONE Cool Gray 11 C` 作为按钮字标深灰参考。模型中的 RGB 只能用于屏幕预览，实体必须用近期 Pantone 色卡和实际材料样片确认。

## 文件

- `cadenza-concept-v1.scad`：可继续调整的参数化源文件
- `cadenza-concept-v1.csg`：OpenSCAD 展开后的几何树，适合交换或排查模型
- `cadenza-concept-v1.stl`：通用网格预览文件
- `cadenza-concept-v1.3mf`：优先用于支持 3MF 的查看器
- `cadenza-concept-v1-preview.png`：快速预览图
- `cadenza-concept-v1-back-pod-preview.png`：一体式实心后壳与顺滑过渡的预览
- `cadenza-concept-v1-front.png`：用于检查屏幕中轴与左右控制区对称关系的正投影视图
- `cadenza-concept-v1-back.png`：用于检查日系仪器式电池/维修盖秩序的背面正投影视图
- `cadenza-concept-v1-side.png`：用于检查“外轮廓内全实体”的楔形侧面
- `cadenza-concept-v1-top.png`：用于检查 USB-C、扬声器防尘网开孔与 `Power/Lock` 的机顶布局
- `cadenza-concept-v1-bottom.png`：用于检查右下角 MicroSD 与两条纵向防滑小棱

## 边界

圆角、壁厚、卡扣、螺钉、按键行程、屏幕固定、音腔、天线净空、连接器安装、公差和材料收缩均未完成工程验证。第三方结构工程师必须以真实器件和制造工艺重新设计。
