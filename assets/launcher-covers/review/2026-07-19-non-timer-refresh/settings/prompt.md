# SETTINGS v1 prompt record

## Identity brief

SETTINGS 是 motion、sound、connectivity 与 Launcher orientation 的控制中心；精确标题 `SETTINGS` 自身成为横跨中央的控制台正面/铭牌，一个旋钮、一个拨杆和一个底部滑块从同一基座上下穿插，明亮、稳定且不采用左字右物。

## Permitted elements

1. 精确标题 `SETTINGS`，只出现一次，并作为 deck 的结构正面。
2. 一块共同基座上的单一 tactile control deck。
3. deck 内恰好一个 rotary dial、一个 physical toggle、一个 horizontal slider；每个至多一个状态标记。

不得增加独立面板、设置列表、Wi-Fi 图标、螺丝、标签、刻度或其他硬件。

## Tonal plan

- 主白块：背景与控件之间的大面积留白。
- 主黑块：完整标题与三个控件的决定性剪影。
- 浅灰面：共同 deck 的顶面，只解释基座关系。
- 深灰面：deck 的正面/侧面，只解释体积和上下穿插。

## Generation record

Built-in `image_gen` used approved TIMER and MOTION masters only to calibrate safe margins, contour confidence, restrained monochrome finish, and reduction discipline. The prompt explicitly prohibited both reference layouts and required a central/full-width faceplate composition with exact spelling `S-E-T-T-I-N-G-S`, no other text, no left-column title, and only the permitted grouped deck.

The original 1886×834 PNG is retained as `source/settings-v1-original.png`; the explicit review master is `source/settings-v1-master-1400x620.png`. Candidate PBMs and previews were generated in `candidate/` with five tonal bands and explicit `--edge-cleanup` only; no formal SETTINGS asset was overwritten.
