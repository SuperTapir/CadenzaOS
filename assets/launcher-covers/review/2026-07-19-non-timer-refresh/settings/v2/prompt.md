# SETTINGS v2 prompt record

## Identity brief

SETTINGS 是一张明亮、权威但亲切的控制中心微型海报：精确标题成为共同控制台的主体，旋钮、拨杆和滑块以硬边插画平面直接表达可调系统，而不是模拟设置列表或棚拍硬件。

## Element and tonal budget

- Exact title: `SETTINGS`, once.
- Dominant motif: one shared control deck containing one knob, one toggle, and one slider.
- Supporting relationship: the title spans the deck face as its central structure.
- Accents: one state mark per control; no other prop.
- Tonal plan: white paper, solid black title/control silhouettes and continuous deck outline, one light top plane, one middle front plane, one small dark control recess plane.

## Generation and targeted edit

The first edit replaced SETTINGS v1's studio product-render language with a flat editorial screenprint / paper-cut technical poster while preserving the central full-width layout. A second targeted edit changed only tonal rendering: all gradients, studio lighting, contact shadows, ambient occlusion, specular highlights and realistic bevel shading were removed in favor of five discrete flat values with crisp boundaries. Exact text, geometry, framing, knob, toggle, slider and continuous outer outline were invariants.

Built-in image generation was used. The generated original is retained as `source/settings-v2-original.png`; the explicit centered 350:155 crop and resize is `source/settings-v2-master-1400x620.png`.
